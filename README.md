# Sales Analysis Using SQL
## Objective: 
Analyze sales data from a used-books company called BookCycle. Identify top-performing products by utilizing common table expressions (CTEs).
## Dataset:
The dataset was pulled from graded lab challenges in the Coursera course, SQL for Data Analysis and Business Insights. This was a course project particularly focused in CTEs and subqueries.
## Key Question Answered:
* What are the top 3 best-selling books per genre in BookCycle's University store location?
## SQL Skills Used:
* Joins
* Window Functions (ROW_NUMBER())
* Subqueries and CTEs
* Aggregations (SUM, COUNT)
* GROUP BY and ORDER BY
* Filters
